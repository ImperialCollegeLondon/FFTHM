# Modified OPM Flow Hysteresis Module (SPHYST)

This repository contains the source files, documentation, and example input files for the modified hysteresis implementation in OPM Flow.

The implementation introduces the `SPHYST` keyword to enable the Spiteri et al. (2005) trapping and hysteresis model.

## Documentation

Detailed implementation, installation instructions, and file modifications are provided in:

docs/SPHYST_OPM_Flow_Implementation_Guide.pdf
